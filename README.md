# EMS, ACLS, PALS, BLS, and NRP Interactive Tutor

A high-fidelity, 3D medical simulation platform for paramedics and PAs.

## Features
- **Compartmental Physiology Engine**: Real-time simulation of interdependent vitals.
- **Interactive 3D Scene**: Click-to-assess patient model with breathing animations.
- **Dynamic ECG Monitor**: Supports 100+ rhythms with realistic morphologies.
- **Gemini AI Feedback**: Clinical debriefing and scoring for every scenario.
- **Multi-Protocol Support**: Dedicated modules for ACLS, BLS, PALS, and NRP.
- **Interactive Anatomy Lessons**: Study pathophysiology with 3D models.

## Getting Started
1. `pnpm install`
2. Create `.env` with `VITE_GEMINI_API_KEY`
3. `npm run dev`
